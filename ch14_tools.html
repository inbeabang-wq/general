Precision Measuring Tools📄 PDF p.416

마이크로미터(Micrometer)는 0.001"(1μm급) 단위, 버니어 캘리퍼(Vernier Caliper)는 0.001"까지 측정합니다. 사용 전 영점(Zero)·교정 상태를 확인합니다.A micrometer measures to 0.001 inch; a vernier caliper reads to 0.001 inch. Check zero and calibration before use.

두께 게이지(Feeler)는 간극, 다이얼 인디케이터는 런아웃(Runout)·편심을 측정합니다. 측정공구는 떨어뜨림·과도한 힘을 피하고 청결히 보관합니다.Feeler gauges measure gaps; dial indicators measure runout/eccentricity. Protect measuring tools from drops and excessive force, and store them clean.

Torque Wrenches📄 PDF p.416

토크렌치는 규정 체결력을 정확히 적용하기 위한 공구입니다. 빔식·클릭식·다이얼식이 있으며, 정기 교정이 필요합니다.A torque wrench applies a precise, specified clamping torque. Beam, click, and dial types exist and require periodic calibration.

공구 관리와 FOD 📗 ASA General

Tool Control & FOD📄 PDF p.416

이물질손상(FOD)은 빠진 공구·부품이 엔진·기구에 들어가 큰 손상을 일으키는 것입니다. 섀도보드(Shadow Board)와 공구 대장으로 작업 전후 공구 수를 대조합니다.Foreign object damage (FOD) occurs when a lost tool/part enters an engine or mechanism. Shadow boards and tool logs are used to account for all tools before and after work.

작업 후 반드시 공구 정수(Inventory)를 확인하고, 분실 공구는 찾을 때까지 항공기를 복귀시키지 않습니다.Always inventory tools after work; if a tool is missing, do not return the aircraft to service until it is found.

📖 핸드북 원문 정리 (FAA-H-8083-30B p.416) ▼
Foreign object damage (FOD) occurs when a lost tool/part enters an engine or mechanism. Shadow boards and tool logs are used to account for all tools before and after work. Always inventory tools after work; if a tool is missing, do not return the aircraft to service until it is found.
이물질손상(FOD)은 빠진 공구·부품이 엔진·기구에 들어가 큰 손상을 일으키는 것입니다. 섀도보드(Shadow Board)와 공구 대장으로 작업 전후 공구 수를 대조합니다.
작업 후 반드시 공구 정수(Inventory)를 확인하고, 분실 공구는 찾을 때까지 항공기를 복귀시키지 않습니다.
